select/option
	-항목 리스트에서 한 개 또는 여러개 선택 가능
	-여러개를 선택 : select 태그 에서 multiple속성
	-화면에 표시할 항목의 갯수 : select 태그 에서 size 속성
	-초기 실행시 표시될 항목 : option 태그에서 selected 속성
	-option태그의 value속성은 서버로 전송되는 값	
	
	-input태그의 value속성은 초기실행시 화면에 표시되는 문자
	-input태그는 한 줄로만 입력이 가능하다
	-여러줄을 입력하기 위해서는 textarea태그 사용
	 가로길이 cols 속성 / 세로길이 rows 속성
가고 싶은 나라(한개 선택)
